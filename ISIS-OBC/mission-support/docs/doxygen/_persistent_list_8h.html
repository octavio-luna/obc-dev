unsigned int  item_size,
unsigned int  max_no_items,
unsigned int  max_iterators,
Boolean  force_empty,
int *  listid 
)

Start an instance of a persistent list.

Parameters
[in]FRAMbaseaddressLowest FRAM address from where to start storing this list
[in]item_sizeSize of the data items in the list, in bytes. This can currently not be larger than XXX bytes
[in]max_no_itemsMaximum number of items that the list can contain
[in]max_iteratorsMaximum number of iterators that can be active for this list
[in]force_emptyForce the list to empty on start, regardless of contents. If there is no list in FRAM then this option not matter, if a list does exist then it will be overwritten with an empty list
[out]listidList ID that needs to be used when referencing this list in other functions in this module
Returns
A negative number on error, 0 when successful
